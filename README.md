# CatWeb

## 介绍

CatWeb一款网页采集器，对网页进行重新排版，以更适合在移动设备上使用

## 测试平台

软件支持跨平台，但仅在一下平台测试过

Android： Android 12 | 小米 11

iOS： iOS 15.4 | iPhone 13

其他平台将在未来更新

## 功能

- [x] 列表，详情，搜索，阅读
